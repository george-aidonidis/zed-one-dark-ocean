# One Dark Ocean for Zed

A dark theme with vivid colors based on [VSCode One Dark Ocean](https://github.com/george-aidonidis/vscode-one-dark-ocean).

## Installation

1. Open Zed
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
3. Search for "zed: extensions"
4. Search for "One Dark Ocean"
5. Click Install

Alternatively, install from the command line:
```bash
zed --install-extension one-dark-ocean
```

## Activation

1. Press `Cmd+K Cmd+T` (Mac) or `Ctrl+K Ctrl+T` (Windows/Linux)
2. Search for "One Dark Ocean"
3. Press Enter to activate

## Development

To test this theme locally:

1. Clone this repository
2. In Zed, press `Cmd+Shift+P` and run "zed: install dev extension"
3. Select the cloned repository directory
4. Activate the theme using the theme selector
